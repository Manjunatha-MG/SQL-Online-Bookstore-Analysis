# 📚 Online Bookstore Analysis using SQL

## 🔍 Project Overview
This project focuses on analyzing an online bookstore dataset using SQL to extract meaningful business insights.

## 🎯 Objectives
- Analyze sales performance
- Identify top-selling books
- Understand customer behavior
- Evaluate revenue trends

## 🛠️ Tools Used
- PostgreSQL
- SQL

## 📂 Dataset
The dataset contains:
- Book details
- Customer information
- Orders and sales data

## 📊 Key Analysis Performed
- Total revenue calculation
- Top-selling books
- Customer purchase patterns
- Monthly sales trends
- Category-wise performance

## 📈 Key Insights
- Certain books contribute majority of revenue
- Repeat customers generate high sales
- Sales increase during specific months
- Some categories perform better than others

## 💡 Skills Demonstrated
- Data querying using SQL
- Aggregations (SUM, COUNT)
- GROUP BY and ORDER BY
- Window functions
- Data analysis and storytelling

## 🚀 Future Improvements
- Connect with Power BI for dashboard
- Add advanced SQL queries
- Include real-time dataset
